# dish.nvim

Not a dish, but a dash. If it was a dish, it would be instant ramen

---

## ‚ú® Features

- Blazingly fast (since its features are minimal)
- Custom ASCII banner on startup
- Only appears when launching Neovim **without arguments**
- Displays the most recently opened buffers with number shortcuts
- Displays a custom list of buffers/projects with letter shortcuts
- Clutterless: Clean and minimal
- Zero dependencies

---

Example installation:
```lua
{
    "CaptainGitHook/dish.nvim",
    config = function()
        require("mydash").setup({
            banner = [[
                Hello there!
            ]],

            -- or optionally:
            -- banner = function()
            --     local s = [[
            --         Hello there!
            --         ]]
            --     return s
            -- end,

            -- Custom banner:
            bannerpath="/path/to/custom/banner.sh",
            
            -- Pin custom files and folders to your dashboard
            projpaths={
                "~/.config/nvim/",
                "/Highly/frequented/folder/2"
            },

            -- Number of recent files pinned to dashboard
            N_RECENT=5
        }
    })
    end,
}
```
